# FastyBird Web UI library

FastyBird Web UI library is a design system built with Storybook and Vue.

#### Installation

```
yarn add @fastybird/web-ui-library
```

Import the FastyBird UI main style sheet in your application:

```
@import "~@fastybird/web-ui-library/web-ui-library.css";
```
