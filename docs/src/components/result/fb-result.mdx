## Result API

<div style={{ marginTop: '30px' }}>
	### Attributes

    | Name      | Description            | Type                                                                       | Default   |
    | --------- | ---------------------- | -------------------------------------------------------------------------- | --------- |
    | title     | title of result        | `string`                                                                   | —         |
    | subtitle  | sub title of result    | `string`                                                                   | —         |
    | status    | result status variant  | `success` \| `warning` \| `info` \| `error` \| `403` \| `404` \| `500`     | `info`    |
    | animation | animate status icon    | `boolean`                                                                  | false     |

    ### Slots

    | Name      | Description                  |
    | --------- | ---------------------------- |
    | icon      | content as result icon       |
    | title     | content as result title      |
    | subtitle  | content as result subtitle  |
    | extra     | content as result extra area |

</div>
