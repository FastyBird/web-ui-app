# Badge

A number or status mark on buttons and icons.

<Canvas of={BadgeStories.Component} />

## Playground

<Controls of={BadgeStories.Component} />

## Basic Usage

Displays the amount of new messages.

The amount is defined with value which accepts Number or String.

<Canvas
	withToolbar={true}
	of={BadgeStories.BasicUsage}
/>

## Max Value

You can customize the max value.

The max value is defined by property max which is a Number. Note that it only works when value is also a Number.

<Canvas
	withToolbar={true}
	of={BadgeStories.MaxValue}
/>

## Customizations

Displays text content other than numbers.

When value is a String, it can display customized text.

<Canvas
	withToolbar={true}
	of={BadgeStories.Customizations}
/>

## Red Dot

Use a red dot to mark content that needs to be noticed.

Use the property `is-dot`. It is a Boolean.

<Canvas
	withToolbar={true}
	of={BadgeStories.RedDot}
/>
