## Divider API

<div style={{ marginTop: '30px' }}>
	### Attributes

    | Name             | Description                                                | Type                                                                                                                                 | Default    |
    | ---------------- | ---------------------------------------------------------- | ------------------------------------------------------------------------------------------------------------------------------------ | ---------- |
    | direction        | set divider's direction                                    | `horizontal` \| `vertical`                                                                                                           | horizontal |
    | border-style     | set the style of divider                                   | `none` \| `solid` \| `hidden` \| `dashed` \| ... [css/border-style](https://developer.mozilla.org/en-US/docs/Web/CSS/border-style)   | solid      |
    | content-position | the position of the customized content on the divider line | `left` \| `right` \| `center`                                                                                                        | center     |
    | gradient         | determine whether show divider with gradient effect        | `boolean`                                                                                                                            | false      |

    ### Slots

    | Name    | Description                            |
    | ------- | -------------------------------------- |
    | default | customized content on the divider line |

</div>
