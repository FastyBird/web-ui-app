# Input Number

Allow users to input numerical values within a customizable range using the `min` and `max` properties of the input component. These properties define
the minimum and maximum values allowed for input, enabling users to specify values within the specified range.

## Basic usage

> **TIP:**
> If an invalid string is entered into the input box, the input value will emit `NaN` to the upper layer as a result of the error.

<Canvas
	withToolbar={true}
	of={InputNumberStories.BasicUsage}
/>

## Disabled

The `disabled` property accepts a `boolean`. If set to `true`, the component is disabled. To control the value within a range, you can use the `min`
property to set the minimum value and `max` to set the maximum value. By default, the minimum value is `0`.

<Canvas
	withToolbar={true}
	of={InputNumberStories.Disabled}
/>

## Steps

You can define incremental steps by adding the `step` property and setting the desired step value.

<Canvas
	withToolbar={true}
	of={InputNumberStories.Steps}
/>

## Step Strictly

The `step-strictly` property accepts a boolean value. When set to true, the input value can only be a multiple of the step value.

<Canvas
	withToolbar={true}
	of={InputNumberStories.StrictlyStep}
/>

## Precision

The `precision` property allows you to specify the precision of the input value.

> **TIP:**
> The value of precision must be a non-negative integer and should not be less than the number of decimal places in the step.

<Canvas
	withToolbar={true}
	of={InputNumberStories.Precision}
/>

## Size

You can use the `size` property to adjust the size of the input field. Possible values include `large` or `small`.

<Canvas
	withToolbar={true}
	of={InputNumberStories.Size}
/>

## Controls Position

You can specify the position of control buttons using the `controls-position` property.

<Canvas
	withToolbar={true}

## Input Number API

<div style={{ marginTop: '30px' }}>
	### Attributes

    | Name                    | Description                                      | Type                                         | Default   |
    | ----------------------- | ------------------------------------------------ | -------------------------------------------- | --------- |
    | model-value / v-model   | binding value                                    | `number`                                     | —         |
    | min                     | the minimum allowed value                        | `number`                                     | -Infinity |
    | max                     | the maximum allowed value                        | `number`                                     | Infinity  |
    | step                    | incremental step                                 | `number`                                     | 1         |
    | step-strictly           | whether input value can only be multiple of step | `boolean`                                    | false     |
    | precision               | precision of input value                         | `number`                                     | —         |
    | size                    | size of the component                            | `large` \| `default` \| `small`              | `default` |
    | readonly ^(2.2.16)      | same as `readonly` in native input               | `boolean`                                    | false     |
    | disabled                | whether the component is disabled                | `boolean`                                    | false     |
    | controls                | whether to enable the control buttons            | `boolean`                                    | true      |
    | controls-position       | position of the control buttons                  | `left` \| `right`                            | `left`    |
    | name                    | same as `name` in native input                   | `string`                                     | —         |
    | label                   | same as `label` in native input                  | `string`                                     | —         |
    | placeholder             | same as `placeholder` in native input            | `string`                                     | —         |
    | id                      | same as `id` in native input                     | `string`                                     | —         |
    | value-on-clear          | value should be set when input box is cleared    | `number` \| `null` \| `(min \| max)`         | —         |
    | validate-event          | whether to trigger form validation               | `boolean`                                    | true      |

    ### Events

    | Name   | Description                     | Type                                                                         |
    | ------ | ------------------------------- | ---------------------------------------------------------------------------- |
    | change | triggers when the value changes | `(currentValue: number \| undefined, oldValue: number \| undefined) => void` |
    | blur   | triggers when Input blurs       | `(event: FocusEvent) => void`                                                |
    | focus  | triggers when Input focuses     | `(event: FocusEvent) => void`                                                |

    ### Exposes

    | Name  | Description                      | Type         |
    | ----- | -------------------------------- | ------------ |
    | focus | get focus the input component    | `() => void` |
    | blur  | remove focus the input component | `() => void` |

</div>
