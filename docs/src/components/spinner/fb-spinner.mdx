# Spinner

A spinner for displaying loading state of a page or a section.

<Canvas of={SpinnerStories.Component} />

## Playground

<Controls of={SpinnerStories.Component} />

## Sizes

Besides default size, Spinner component provides three additional sizes for you to choose among different scenarios.

Use property `size` to set additional sizes with `large`, `small`.

<Canvas of={SpinnerStories.Sizes} />

## Variants

Use property `variant` to set specific variant with `primary`, `info`, `success`, `warning`, `danger`.

<Canvas of={SpinnerStories.Variants} />

## Spinner API

<div style={{ marginTop: '30px' }}>
	### Attributes

    | Name              | Description                                                   | Type                                                                    | Default              |
    | ----------------- | ------------------------------------------------------------- | ----------------------------------------------------------------------- | -------------------- |
    | size              | spinner size                                                  | `large` \| `default` \| `small`                                         | `default`            |
    | variant           | spinner variant                                               | `primary` \| `default` \| `info` \| `success` \| `warning` \| `danger`  | `default`            |
    | bg                | determine whether the spinner background color is always on   | `boolean`                                                               | true                 |

</div>
