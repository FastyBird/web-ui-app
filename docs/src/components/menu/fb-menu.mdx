# Menu

Navigation menu for website navigation.

## Top Bar

The Top Bar Menu is versatile and adaptable to various scenarios.

By default, the Menu is vertical, but you can switch it to horizontal mode by setting the `mode` property to `horizontal`. Additionally, you can utilize
the sub-menu component to create a second-level menu. Customize the colors of the Menu using `background-color`, `text-color`, and `active-text-color` properties.

<Canvas of={MenuStories.TopBar} />

## Left And Right

You can position the menu items to the left or right as needed.

<Canvas of={MenuStories.LeftRight} />

## Side Bar

Vertical Menu with Submenus:

Utilize the `fb-menu-item-group` component to establish a menu group. The title of the group can be specified using the `title` property or a named slot.

<Canvas of={MenuStories.SideBar} />

## Collapse

The Vertical Menu can be collapsed to save space and provide a more concise navigation experience.

<Canvas of={MenuStories.Collapse} />

## Popper Offset

When using the `popper-offset` property on the Menu, it will override the Submenu's popper-offset settings.

## Menu API

<div style={{ marginTop: '30px' }}>

    ## Attributes

    | Name                     | Description                                                                                                                                                           | Type                        | Default    |
    | ------------------------ | --------------------------------------------------------------------------------------------------------------------------------------------------------------------- | --------------------------- | ---------- |
    | mode                     | menu display mode                                                                                                                                                     | `horizontal` \| `vertical`  | `vertical` |
    | collapse                 | whether the menu is collapsed (available only in vertical mode)                                                                                                       | `boolean`                   | false      |
    | ellipsis                 | whether the menu is ellipsis (available only in horizontal mode)                                                                                                      | `boolean`                   | true       |
    | ellipsis-icon            | custom ellipsis icon (available only in horizontal mode and ellipsis is true)                                                                                         | `string` \| `Component`     | —          |
    | popper-offset            | offset of the popper (effective for all submenus)                                                                                                                     | `number`                    | 6          |
    | background-color         | background color of Menu (hex format)                                                                                                                                 | string                      | —          |
    | text-color               | text color of Menu (hex format)                                                                                                                                       | string                      | —          |
    | active-text-color        | text color of currently active menu item (hex format)                                                                                                                 | string                      | —          |
    | default-active           | index of active menu on page load                                                                                                                                     | `string`                    | —          |
    | default-openeds          | array that contains indexes of currently active sub-menus                                                                                                             | `Array`                     | —          |
    | unique-opened            | whether only one sub-menu can be active                                                                                                                               | `boolean`                   | false      |
    | menu-trigger             | how sub-menus are triggered, only works when `mode` is 'horizontal'                                                                                                   | `hover` \| `click`          | `hover`    |
    | router                   | whether `vue-router` mode is activated. If true, index will be used as 'path' to activate the route action. Use with `default-active` to set the active item on load. | `boolean`                   | false      |
    | collapse-transition      | whether to enable the collapse transition                                                                                                                             | `boolean`                   | true       |
    | popper-effect            | Tooltip theme, built-in theme: `dark` \| `light` when menu is collapsed                                                                                               | `dark` \| `light`           | `dark`     |
    | close-on-click-outside   | optional, whether menu is collapsed when clicking outside                                                                                                             | `boolean`                   | false      |
    | popper-class             | custom class name for all popup menus                                                                                                                                 | `string`                    | —          |
    | show-timeout             | Control timeout for all menus before showing                                                                                                                          | `number`                    | 300        |
    | hide-timeout             | Control timeout for all menus before hiding                                                                                                                           | `number`                    | 300        |

    ## Events

    | Name   | Description                               | Type                                                                                                                     |
    | ------ | ----------------------------------------- | ------------------------------------------------------------------------------------------------------------------------ |
    | select | callback function when menu is activated  | `(index: string, indexPath: string[], item: MenuItemClicked, routerResult?: Promise<void \| NavigationFailure>) => void` |
    | open   | callback function when sub-menu expands   | `(index: string, indexPath: string[]) => void`                                                                           |
    | close  | callback function when sub-menu collapses | `(index: string, indexPath: string[]) => void`                                                                           |

    ## Exposes

    | Method  | Description               | Type                       |
    | ------- | ------------------------- | -------------------------- |
    | open    | open a specific sub-menu  | `(index: string) => void`  |
    | close   | close a specific sub-menu | `(index: string) => void`  |

    ## Slots

    | Name | Description               | Subtags                            |
    | ---- | ------------------------- | ---------------------------------- |
    | —    | customize default content | SubMenu / MenuItem / MenuItemGroup |

</div>

## Sub Menu API

<div style={{ marginTop: '30px' }}>

    ## Attributes

    | Name                  | Description                                                                                                                                   | Type                    | Default                                         |
    | --------------------- | --------------------------------------------------------------------------------------------------------------------------------------------- | ----------------------- | ----------------------------------------------- |
    | index                 | unique identification                                                                                                                         | `string`                | —                                               |
    | popper-class          | custom class name for the popup menu                                                                                                          | `string`                | —                                               |
    | show-timeout          | timeout before showing a sub-menu(inherit `show-timeout` of the menu by default)                                                              | `number`                | —                                               |
    | hide-timeout          | timeout before hiding a sub-menu(inherit `hide-timeout` of the menu by default)                                                               | `number`                | —                                               |
    | disabled              | whether the sub-menu is disabled                                                                                                              | `boolean`               | false                                           |
    | teleported            | whether popup menu is teleported to the body                                                                                                  | `boolean`               | level one SubMenu: true / other SubMenus: false |
    | popper-offset         | offset of the popper (overrides the `popper` of menu)                                                                                         | `number`                | —                                               |
    | expand-close-icon     | Icon when menu are expanded and submenu are closed, `expand-close-icon` and `expand-open-icon` need to be passed together to take effect      | `string` \| `Component` | —                                               |
    | expand-open-icon      | Icon when menu are expanded and submenu are opened, `expand-open-icon` and `expand-close-icon` need to be passed together to take effect      | `string` \| `Component` | —                                               |
    | collapse-close-icon   | Icon when menu are collapsed and submenu are closed, `collapse-close-icon` and `collapse-open-icon` need to be passed together to take effect | `string` \| `Component` | —                                               |
    | collapse-open-icon    | Icon when menu are collapsed and submenu are opened, `collapse-open-icon` and `collapse-close-icon` need to be passed together to take effect | `string` \| `Component` | —                                               |

    ## Slots

    | Name  | Description               | Subtags                            |
    | ----- | ------------------------- | ---------------------------------- |
    | —     | customize default content | SubMenu / MenuItem / MenuItemGroup |
    | title | customize title content   | —                                  |

</div>
