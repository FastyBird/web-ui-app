## Icon Button

Use icons to add more meaning to Button. You can use icon alone to save some space, or use it with text.

Use the `icon` property to add icon. We are using [FontAwesome icons](https://fontawesome.com/icons/). Adding icons to the right side of the text is
achievable with an `<i>` tag. Custom icons can be used as well.

## Button API

<div style={{ marginTop: '30px' }}>
	### Attributes

    | Name              | Description                                                             | Type                                                                      | Default       |
    | ----------------- | ----------------------------------------------------------------------- | ------------------------------------------------------------------------- | ------------- |
    | size              | button size                                                             | `large` \| `default` \| `small`                                           | `default`     |
    | variant           | button variant                                                          | `primary` \| `default` \| `info` \| `success` \| `warning` \| `danger`    | —             |
    | type              | same as native button's `type`                                          | `button` \| `submit` \| `reset`                                           | `button`      |
    | block             | whether button is stretched to full width                               | `boolean`                                                                 | false         |
    | uppercase         | whether button text is in uppercase                                     | `boolean`                                                                 | false         |
    | link              | whether it's a link button                                              | `boolean`                                                                 | false         |
    | round             | whether it's a round button                                             | `boolean`                                                                 | false         |
    | circle            | whether it's a circle button                                            | `boolean`                                                                 | false         |
    | text              | whether it's a text button                                              | `boolean`                                                                 | false         |
    | plain             | whether it's a plain button                                             | `boolean`                                                                 | false         |
    | bg                | whether the text button background color is always on                   | `boolean`                                                                 | false         |
    | color             | custom button color, automatically calculate `hover` and `active` color | `string`                                                                  | —             |
    | active            | whether it's a active button                                            | `boolean`                                                                 | false         |
    | disabled          | whether button is disabled                                              | `boolean`                                                                 | false         |
    | autofocus         | same as native button's `autofocus`                                     | `boolean`                                                                 | false         |
    | loading           | whether it's loading                                                    | `boolean`                                                                 | false         |
    | loading-icon      | customize loading icon component                                        | `string` \| `Component`                                                   | —             |
    | icon              | icon component                                                          | `string` \| `Component`                                                   | —             |
    | tag               | custom element tag                                                      | `string` \| `Component`                                                   | `button`      |
    | dark              | dark mode, which automatically converts `color` to dark mode colors     | `boolean`                                                                 | false         |

    ### Slots

    | Name    | Description                                 |
    | ------- | ------------------------------------------- |
    | default | required slot representing button content   |
    | loading | customize loading component                 |
    | icon    | customize icon component                    |

    ### Exposes

    | Name           | Description          | Type                                      |
    | -------------- | -------------------- | ----------------------------------------- |
    | ref            | button html element  | `HTMLButtonElement`                       |
    | size           | button size          | `ComputedRef<large \| default \| small>`  |
    | disabled       | button disabled      | `ComputedRef<boolean>`                    |

</div>
