# Icon With Child

The dividing line that separates the content.

## Basic Usage

The child icons offer support for basic theme variants, which can be set using the `variant` property. Icons can be inserted using the predefined slots.

<Canvas of={IconWithChildStories.BasicUsage} />

## Sizes

Similar to the `fb-icon` component, this component also supports different icon sizes. You only need to specify the main icon size using the `size` property,
and the size of the child icon will be calculated automatically.

<Canvas of={IconWithChildStories.Sizes} />

## Icon With Child API

<div style={{ marginTop: '30px' }}>
	### Attributes

    | Name        | Description             | Type                                                      | Default  |
    | ----------- | ----------------------- |-----------------------------------------------------------| -------- |
    | variant     | child icon variant      | `default` \| `info` \| `success` \| `warning` \| `danger` | `default`|
    | size        | main icon size          | `number`                                                  | —        |
    | color       | main icon color         | `string`                                                  | —        |

    ### Slots

    | Name     | Description                                 |
    | -------- |-------------------------------------------- |
    | icon     | slot for inserting main icon.               |
    | child    | slot for inserting child icon.              |

</div>
