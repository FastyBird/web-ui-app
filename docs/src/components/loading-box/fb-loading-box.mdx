## Loading Box API

<div style={{ marginTop: '30px' }}>
	### Attributes

    | Name              | Description                                                             | Type                                                                     | Default       |
    | ----------------- | ----------------------------------------------------------------------- | ------------------------------------------------------------------------ | ------------- |
    | size              | loading box size                                                        | `large` \| `default` \| `small`                                          | `default`     |
    | variant           | loading box variant                                                     | `primary` \| `default` \| `info` \| `success` \| `warning` \| `danger`   | `default`     |
    | show              | determine whether loading box is shown                                  | `boolean`                                                                | false         |
    | full-screen       | determine whether loading box is stretched to full screen               | `boolean`                                                                | false         |
    | animation         | determine whether loading box is shown with animation effect            | `boolean`                                                                | false         |
    | spinner           | customize loading box spinner component                                 | `string` \| `boolean` \| `Component`                                     | true          |

    ### Slots

    | Name    | Description                                      |
    | ------- | ------------------------------------------------ |
    | default | optional slot representing loading box content   |
    | icon    | customize icon component                         |

</div>
