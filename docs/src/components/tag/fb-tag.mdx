# Tag

This component is used for marking and selection purposes.

## Basic usage

You can define the color variant of the Tag using the `variant` property. Additionally, you can set the background color of the Tag using the `color` property.

<Canvas of={TagStories.BasicUsage} />

## Removable

You can make a Tag removable by setting the `closable` property to `true`. By default, the removal of the Tag includes a fading animation. If you prefer
to disable the animation, you can set the `disable-transitions` property to `true`. The `close` event is triggered when the Tag is removed.

<Canvas of={TagStories.Removable} />

## Edit Dynamically

You can dynamically add or remove tags using the `close` event.

<Canvas of={TagStories.Dynamical} />

## Sizes

The Tag component offers three additional sizes besides the default size, allowing you to choose the most suitable one for different scenarios. You can
set the size using the `size` property with options `large`, `default`, or `small`.

<Canvas of={TagStories.Sizes} />

## Theme

The Tag component offers three distinct themes: `dark`, `light`, and `plain`.

Use the `effect` property to switch between them, with `light` being the default.

<Canvas of={TagStories.Theme} />

## Rounded

Tags can also be rounded like buttons.

<Canvas of={TagStories.Rounded} />

## Checkable

At times, due to business requirements, we may need a tag that resembles a checkbox. However, a button-like checkbox cannot meet these needs. This is
where the check-tag component comes into play. You can use the `type` property to achieve this.

<Canvas of={TagStories.Checkable} />

## Tag API

<div style={{ marginTop: '30px' }}>
	### Attributes

    | Name                | Description                          | Type                                                          | Default   |
    | ------------------- | ------------------------------------ | ------------------------------------------------------------- | --------- |
    | type                | type of Tag                          | `primary` \| `success` \| `info` \| `warning` \| `danger`     | `primary` |
    | closable            | whether Tag can be removed           | `boolean`                                                     | false     |
    | disable-transitions | whether to disable animations        | `boolean`                                                     | false     |
    | hit                 | whether Tag has a highlighted border | `boolean`                                                     | false     |
    | color               | background color of the Tag          | `string`                                                      | —         |
    | size                | size of Tag                          | `large` \| `default` \| `small`                               | —         |
    | effect              | theme of Tag                         | `dark` \| `light` \| `plain`                                  | `light`   |
    | round               | whether Tag is rounded               | `boolean`                                                     | false     |

    ### Events

    | Name  | Description                  | Type                     |
    | ----- | ---------------------------- | ------------------------ |
    | click | triggers when Tag is clicked | `(evt: UiEvent) => void` |
    | close | triggers when Tag is removed | `(evt: UiEvent) => void` |

    ### Slots

    | Name    | Description               |
    | ------- | ------------------------- |
    | default | customize default content |

</div>
